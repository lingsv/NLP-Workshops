# :woman: :woman: Workshop for Her 2020 :muscle: ♀️

Com a pandemia de COVID-19, essa edição é online.
Esse evento é realizado pelo [Grupo Turing](grupoturing.com.br), [Elas pleas Exatas]() e [Data Girls]().
